# Bin2Dec

A simple Java console application that converts a binary number into its decimal equivalent.

## Description

This project was created as a beginner programming exercise based on the **Bin2Dec** challenge from App Ideas.

The program:

- Accepts a binary number from the user.
- Validates that the input contains only `0` and `1`.
- Converts the binary number to decimal.
- Displays the result.

## Features

- ✅ Accepts binary input
- ✅ Input validation
- ✅ Converts binary to decimal
- ✅ Supports up to 8 binary digits (or more if you implemented the bonus)

## Example

```text
Enter a binary number:
101101

Decimal: 45
```

Invalid input:

```text
Enter a binary number:
10201

Error: Binary numbers may only contain 0 and 1.
```

## Concepts Practiced

- Java loops
- Strings
- Input validation
- Mathematical operations (`Math.pow()`)
- Conditionals

## How to Run

Compile:

```bash
javac Main.java
```

Run:

```bash
java Main
```
